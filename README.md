# Shake Me
My kids wanted to see something that changed the color of the screen when they shake the phone/tablet (also works on my convertible laptop since it can double as a tablet).

Doesn't work on Apple devices cuz they locked down motion sensing for PWAs and I didn't want to bother messing with the workarounds. You can still tap the screen and it'll work.
